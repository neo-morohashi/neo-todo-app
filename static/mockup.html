1. カテゴリ → タグへの移行
・サイドバーのタグ一覧で絞り込み（#マーサーをクリックすると配下のタスクが出る）
・タスク行にタグpillを表示。1タスクに複数タグOK（例: #マーサー + #Mercer-AI）
・既存のカテゴリは全てタグ名に自動変換（マーサー → #マーサー、Mercer-AI → #Mercer-AI...）
・親子関係（マーサー > Mercer-AI）は サイドバーのインデント表示で表現（タグ自体はフラット）
2. 2段階削除フロー
通常時：行末に ×（hover時のみ表示） → クリックで横線（pending-delete）状態に → ↶ 戻す または 🗑 完全削除
・pending-delete状態は薄ピンク背景＋横線で「削除予約中」を視覚化
・↶ で取り消し（誤操作防止）／ 🗑 で確定削除（モーダル確認なし）
・既存の「完了（◯チェック）」とは独立。完了は archive.md 行き、削除は完全消去
3. タスク追加時のタグ入力（イメージ）
モーダル内で「タスク名」フィールドに # を入力すると候補がサジェストされる:
交通費精算 #マ | #マーサー #Mercer-Admin
